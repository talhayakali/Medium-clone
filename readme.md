# Medium-Clone Project

A static front-end clone of [Medium](https://medium.com/membership) built with html, css and bootstrap 5.

## Pages

- `index.html` — Home page
- `membership.html` — Membership plans page
- `ourstory.html` — Our story page


## Technologies

- HTML5
- Bootstrap 5.3.8
- Font Awesome 6.5.1
- CSS3
